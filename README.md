## Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yarik-osg/RecMan.git
   cd RecMan
   ```

2. **Navigate to the application:**
   ```bash
   cd my-app
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
